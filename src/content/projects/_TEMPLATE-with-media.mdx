## An interactive embed

Point at something hosted elsewhere (an Observable notebook, a Plotly/Streamlit
app, a YouTube video…). It loads in a sandboxed, lazy iframe. `title` is
required. Delete this block if unused.

<Embed
  src="https://example.org/your-hosted-interactive-thing"
  title="Short description of the interactive content"
  ratio="16/9"
/>

## A static figure

For a chart or diagram exported as an image. `alt` is required; `caption` is
optional. Put the image under `public/images/` or use a full URL.

<Figure
  src="/images/my-project/results.svg"
  alt="Describe what the chart shows, for screen readers"
  caption="Fig 1. A short caption."
/>
